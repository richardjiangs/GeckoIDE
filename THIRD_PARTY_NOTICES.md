# Third-Party Notices

## Pyodide

The Android APK bundles Pyodide `0.29.4` runtime files so Python can run offline inside the app:

- `pyodide.js`
- `pyodide.asm.wasm`
- `python_stdlib.zip`
- `pyodide-lock.json`
