Mac

Download the `.command` file with the hover scrollbar and Outline navigator, then run the Terminal commands below.

Download GeskoIDE.command 
cd ~/Downloads
chmod +x "GeskoIDE.command"
xattr -d com.apple.quarantine "GeskoIDE.command"
./"GeskoIDE.command"
